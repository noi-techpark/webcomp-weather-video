## Usage

Build all widget using Rollup:

```
yarn build
```

Watch component using Rollup with dev purpose:

```
yarn watch
```

To view the component changes:

```
cd ./work
sh ./work/serve.sh
```

You will see the components in action at [http://0.0.0.0:8000/](http://0.0.0.0:8000/) url.
